Используется для встраивания фрейма, который позволяет загружать любые другие документы.

Так как элемент <iframe> является контейнером, то его содержание может игнорироваться браузерами, который данный элемент не поддерживают. Рекомендуется для таких случаев использовать альтернативный текст, который должен располагаться внутри элемента <iframe>.
